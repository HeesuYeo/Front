# Float

  • 다른 내용들이 해당 요소를 비켜가게 만든다. (시냇가에 커다란 돌 같은 느낌)
  • clear 속성을 사용하면 clear를 설정한 해당 요소 옆에 다른 Float 요소가 올 수 없도록 막을 수 있다.
  • 요소 내부에서 float을 사용한 경우 float 요소에 대한 높이를 인식하지 못하는 문제를 clear:both를 통해 해결해 줄 수 있다.
